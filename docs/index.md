## 目标

当前社区有很多 React 源码分析文章，但不够成体系，内容质量也参差不齐。基于此原因，这本电子书的目标是打造一本成体系、高质量的React源码电子书。为了达到这个目标，在行文上，本书会遵循：

1. 不预设观点。所有观点来自React核心团队成员在公开场合发表的内容。
2. Demo理解。对每个知识点提供Demo辅助理解。
3. 保持更新。在React版本更新后会及时补充。当前版本`v16.13.1`
